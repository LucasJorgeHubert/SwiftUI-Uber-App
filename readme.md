### Setup

1. Clone this project in your local machine
2. Open Project in XCode
3. Run project

### App Features

- List map locations
- Dark mode support
- Create a line to show route
- Compute travel time
- Ride screen details
- Ride options
- Calcule fare trip

### App definitions

- MVVM archtecture
- Local assets
- Min. target = 16.0
- Request user authorization to get localization data
